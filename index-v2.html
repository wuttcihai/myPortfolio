About Me

สวัสดีครับ ผมชื่อ ล้อ'แม็ค

ผมเป็นนักพัฒนาซอฟต์แวร์สาย Full-Stack ที่มีประสบการณ์พัฒนาระบบทั้ง Web Application, Windows Application, Windows Service และ RESTful API โดยให้ความสำคัญกับโครงสร้างระบบที่ดี ความเสถียร และประสิทธิภาพในการใช้งานจริง

มีความถนัดทั้งฝั่ง Front-end และ Back-end สามารถออกแบบและพัฒนาระบบตั้งแต่การวิเคราะห์ความต้องการ ออกแบบฐานข้อมูล ไปจนถึงการเชื่อมต่อระบบและนำขึ้นใช้งานจริง รองรับการขยายระบบในอนาคต

ผมมีประสบการณ์ทำงานกับฐานข้อมูลหลากหลายรูปแบบ เช่น MySQL, SQL Server และ MongoDB รวมถึงการพัฒนา API เพื่อเชื่อมต่อกับระบบภายนอก และการสร้าง Service สำหรับทำงานเบื้องหลังอัตโนมัติ
